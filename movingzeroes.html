Moving zeroes

Explanation:

This problem can be looked at from the point of a stack. Meaning you iterate from left to right(top to bottom) and then you pop any zeroes but make sure as you remove them you add them to the array (top of the stack).

Python code

    class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        for i in range(len(nums)-1, -1, -1):
            if nums[i] == 0:
                nums.pop(i)
                nums.append(0)
